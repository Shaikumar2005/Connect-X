Available Forms

📝 Instructions

1. Bonafide Certificate

  • Enter your full name as per college records.
  • Do get Signature from HOD
  • Submit to: Admin Office - Main Block GND Floor

2. Leave Application Form

  • Provide valid reason for leave.
  • Attach medical certificate if applicable.
  • Submit to: Class Advisor

3. ID Card Request Form

  • Upload/attach passport-size photo.
  • Include enrollment number correctly.
  • Submit to: Admin Block – Main Block GND Floor

3. IDEA LAB Request Form

  • Get a Form from DEO IT
  • Fill it and get signatures from Authorities mentioned in FORM
  • Submit to: IDEA LAB I Incharge
